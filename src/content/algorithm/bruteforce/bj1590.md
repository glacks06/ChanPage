각 버스에 대해
버스 탑승 시간의 최솟값을 res로 두면
도착시간이 버스 시작시간이면 res = 0
도착 시간이 버스 시작 시간 전이면 res = (버스 시작시간) - (도착시간)
도착 시간이 마지막 버스 시간 후이면 continue
이 모든것이 아니면 res = 도착 후 제일 먼저 오는 버스

각 버스에 대한 res들 중 최솟값 저장 후 최솟값이 초기화한 값(Int범위 중 가장 큰 값)이면 버스가 없음을 의미한다.

C

```c
#include <stdio.h>

int main(){
    int n, t; scanf("%d %d", &n, &t);
    int min = 2147483647;
    for(int i = 0; i < n; i++){
        int s, l, c; scanf("%d %d %d", &s, &l, &c);
        int res = 2147483647;

        if(t <= s) res = s - t;
        else {
            long long last_bus = (long long)s + (long long)l * (c - 1);
            if(t <= last_bus) {
                int wait = (t - s) % l;
                if(wait == 0) res = 0;
                else res = l - wait;
            } else continue;
        }
        if(res < min) min = res;
    }
    if(min == 2147483647) printf("-1");
    else printf("%d", min);
    return 0;
}
```

Python

```python
import sys

n, t = map(int, sys.stdin.readline().split())
min_val = 2**31

for _ in range(n):
    s, l, c = map(int, sys.stdin.readline().split())
    last_bus = s + l * (c - 1)
    
    if t <= s:
        res = s - t
    elif t <= last_bus:
        wait = (t - s) % l
        res = 0 if wait == 0 else l - wait
    else:
        continue
        
    if res < min_val:
        min_val = res

if min_val == 2**31:
    print("-1")
else:
    print(min_val)
```
