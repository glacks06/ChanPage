[https://www.acmicpc.net/problem/11060](https://www.acmicpc.net/problem/11060)

1xN크기의 미로에서 칸에 쓰여진 수 이하 만큼 점프를 할 수 있고, 오른쪽 끝 칸까지 최소 몇번 점프해야 하는지 구하는 문제이다.
첫 칸부터 경우의 수를 계산해 나가며 최소 값을 도출해야하며, 이 과정에서 DP를 사용할 수 있다.

d[i]를 i번째 칸까지 최소로 올 수 있는 점프 횟수라고 두었다.
그 다음 첫번째 칸에서부터 탐색을 시작하는데, 현재의 칸(d[j]라고 가정.)에 적힌 수만큼 앞 칸(d[i]라고 가정.)을 탐색한다.
이때 값이 한번도 갱신이 되지 않은 상태(즉, 한번도 탐색을 진행하지 않은 상태)라면
d[i] = d[j]+1
이고, 이미 탐색이 진행되었던 칸이지만 d[i]가 d[j]+1보다 작으면(= 이전에 저장한 최소 점프 횟수보다 j번째 칸에서 점프했을 때 더 최소로 점프하게 된다면)
d[i] = d[j]+1
위와 같이 점화식을 작성할 수 있다.

탐색을 진행하던 중 현재 위치에서 점프하여 가장 오른쪽 칸에 도달 가능하다면 그 값을 저장하여 결과로 출력한다.

C

```c
#include <stdio.h>

int a[1000];
int d[1000];

int res_func(int n){
    int res = 2100000000;
    for(int i = 0; i < n; i++){
        if(i != 0 && d[i] == 0) continue;
        for(int j = i+1; j < i+a[i]+1; j++){
            if(j >= n-1){
                if(d[i]+1 < res) res = d[i]+1;
            }
            else if(d[j] == 0 || d[i]+1 < d[j]) d[j] = d[i] + 1;
        }
    }
    return (res == 2100000000) ? -1 : res;
}

int main() {
    int n; scanf("%d", &n);
    for(int i = 0; i < n; i++)
        scanf("%d", &a[i]);
    if(n == 1) printf("0");
    else printf("%d", res_func(n));
    return 0;
}
```

Python

```python
import sys

def res_func():
    res = 2100000000
    for i in range(n):
        if i != 0 and d[i] == 0: continue
        for j in range(i+1, i+a[i]+1):
            if j >= n-1: 
                if d[i]+1 < res:
                    res = d[i]+1
            elif d[j] == 0 or d[i]+1 < d[j]:
                d[j] = d[i]+1
    
    return -1 if res == 2100000000 else res

n = int(sys.stdin.readline())
a = list(map(int, sys.stdin.readline().split()))

d = [0 for _ in range(n)]
d[0] = 0

if n == 1:
    print(0)
else:
    print(res_func())
```
