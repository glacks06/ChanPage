반복문을 통해 첫째 줄에는 공백 N-1개와 별 1개, 둘째 줄에는 공백 N-2개와 별 2개, 셋째줄에는 공백 N-3개와 별 3개... N번째 줄에는 별 2xN - 1개를 출력하는 형식에서 (각 줄의 별을 출력하는 개수) - 1 만큼 추가적으로 출력하는 방식으로 문제를 해결하였다.

C

```c
#include <stdio.h>

int main() {
    int n; scanf("%d", &n);
    for(int i = 1; i <= n; i++){
        for(int j = 1; j <= n-i; j++) printf(" "); // 공백
        for(int j = 1; j <= i; j++) printf("*"); // 별1
        for(int j = 1; j <= i-1; j++) printf("*"); // 별2
        printf("\n");
    }
}
```

Python

```python
n = int(input())
for i in range(1, n+1):
    print(' ' * (n-i), end='')
    print('*' * i, end='')
    print('*' * (i-1))
```

Java

```java
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      int n = s.nextInt();
      
      for(int i = 1; i <= n; i++){
          for(int j = n-i; j > 0; j--) System.out.print(" ");
          for(int j = 1; j <= i; j++) System.out.print("*");
          for(int j = 1; j < i; j++) System.out.print("*");
          System.out.print("\n");
      }
    }
}
```
