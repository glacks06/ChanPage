n*n크기의 배열을 만든 후(table)
가장 왼쪽 위부터 반시계 방향으로 숫자를 채워나갔다. 이 과정에서 이중 반복문을 사용하였다. 반시계방향으로 한바퀴를 돌면 바로 아래의 칸부터 다시 반시계방향으로 채워나간 것이다. 숫자는 n*n부터 1까지 줄어든다.

출력 단계에서 해당 2차원 배열을 출력하면서 주어진 n값과 동일한 값을 가진 인덱스를 확인하고, 이를 변수(x, y)에 저장해두었다가 마지막에 출력하였다.

C

```c
#include <stdio.h>

int main(){
    int n, m;
    scanf("%d %d", &n, &m);
    int table[n][n];
    int v = n * n;
    for(int i = 0; i < 1 + (n / 2); i++){
        for(int j = i; j < n - i; j++) table[i][j] = v--;
        for(int j = i + 1; j < n - i; j++) table[j][((n-1) - i)] = v--;
        for(int j = i + 1; j < n - i; j++) table[(n-1) - i][(n-1) - j] = v--;
        for(int j = i + 1; j < n - i - 1; j++) table[(n-1) - j][i] = v--;
    }

    int x, y;
    for(int i = 0; i < n; i++){
        for(int j = 0; j < n; j++){
            printf("%d ", table[j][i]);
            if(table[j][i] == m) {x = i + 1; y = j + 1;}
        }
        printf("\n");
    }
    printf("%d %d", x, y);
}
```

Python

```python
n = int(input()); m = int(input())
table = [[0 for _ in range(n)] for _ in range(n)]
v = n ** 2

for i in range(1 + (n // 2)):
    for j in range(i, n - i): table[i][j] = v; v -= 1
    for j in range(i + 1, n - i): table[j][(n-1) - i] = v; v -= 1
    for j in range(i + 1, n - i): table[(n-1) - i][(n-1) - j] = v; v -= 1
    for j in range(i + 1, n - i - 1): table[(n-1) - j][i] = v; v -= 1

for i in range(n):
    for j in range(n):
        print(table[j][i], end=' ')
        if table[j][i] == m: x = i + 1; y = j + 1
    print()

print(x, y)
```
