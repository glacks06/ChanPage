Rev함수제작:
역수로 만들 값을 담은 변수(v)
역수가 된 값을 저장할 변수(res)
res에 v의 일의자리 숫자를 저장하고 10을 곱한 후 v를 10으로 나눈다(몫만 구함)
이 과정을 v가 0이 될 때까지 반복한다.

앞서 만든 Rev를 이용해 Rev(Rev(x) + Rev(y)) 구하는 문제

C

```c
#include <stdio.h>

int rev(int v){
    int res = 0;
    while(v){
        res *= 10;
        res += v % 10;
        v /= 10;
    }
    return res;
}

int main() {
    int x, y; scanf("%d %d", &x, &y);
    printf("%d", rev(rev(x) + rev(y)));
}
```

Python

```python
def rev(v):
    res = 0
    while v:
        res *= 10
        res += v % 10
        v //= 10
    return res

x, y = map(int, input().split())
print(rev(rev(x) + rev(y)))
```
