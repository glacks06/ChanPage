반복문을 통해 첫째줄에는 공백을 0번 별을 N번, 둘째줄에는 공백을 1번 별을 N-1번, 셋째줄에는 공백을 2번 별을 N-2번... N번째줄에는 공백을 N-1번 별을 1번 출력하도록 하여 문제를 해결하였다.

C

```c
#include <stdio.h>

int main() {
    int n; scanf("%d", &n);
    for(int i = n; i >= 1; i--){
        for(int j = 1; j <= n-i; j++) printf(" "); // 공백
        for(int k = i; k >= 1; k--) printf("*"); // 별
        printf("\n");
    }
}
```

<br>

Python

```python
n = int(input())
for i in range(n):
    print(' ' * i, end='')
    print('*' * (n-i))
```

Java

```java
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      int n = s.nextInt();
      
      for(int i = n; i > 0; i--){
          for(int j = 0; j < n-i; j++) System.out.print(" ");
          for(int j = 1; j <= i; j++) System.out.print("*");
          System.out.print("\n");
      }
    }
}
```
