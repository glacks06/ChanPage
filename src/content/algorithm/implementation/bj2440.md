[https://www.acmicpc.net/problem/2440](https://www.acmicpc.net/problem/2440)

반복문을 통해 첫째줄에는 별을 N번, 둘째줄에는 별을 N-1번, 셋째줄에는 별을 N-3번... N번째 줄에는 별을 1번 출력하도록 하여 문제를 해결하였다.

C

```c
#include <stdio.h>

int main() {
    int n; scanf("%d", &n);
    for(int i = n; i >= 1; i--){
        for(int j = i; j >= 1; j--) printf("*");
        printf("\n");
    }
}
```

<br>

Python

```python
n = int(input())
for i in range(n): print('*' * (n-i))
```

<br>

Java

```java
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      int n = s.nextInt();
      
      for(int i = n; i > 0; i--){
          for(int j = 1; j <= i; j++) System.out.print("*");
          System.out.print("\n");
      }
    }
}
```
