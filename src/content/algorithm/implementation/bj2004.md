조합은 위와 같은 수식으로 나타낼 수 있다.

이때 뒷자리 0의 개수는 팩토리얼 0의 개수때와 같이 결과를 소인수분해 하였을 때 2와 5의 개수를 확인하면 된다.

따라서 약분 후에 2나 5의 개수를 확인한 후 더 작은 값을 출력하여서 문제를 해결했다.


C

```c
#include <stdio.h>

long long count(int n, int k) {
    long long cnt = 0;
    for (long long i = k; i <= n; i *= k) {
        cnt += n / i;
    }
    return cnt;
}

int main() {
    int n, m;
    scanf("%d %d", &n, &m);

    long long cnt2 = count(n, 2) - count(m, 2) - count(n - m, 2);
    long long cnt5 = count(n, 5) - count(m, 5) - count(n - m, 5);

    printf("%lld\n", cnt2 < cnt5 ? cnt2 : cnt5);

    return 0;
}
```
