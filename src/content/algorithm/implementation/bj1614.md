각각 손가락을 12345로 넘버링
123454321.... 순서로 확인하면서 다친 손가락이 나올 때 마다
사용가능 횟수 차감
사용 횟수 0이 된 후 다친 손가락이 나올때까지 카운팅
결과 출력

내코드
12345432 이 수열 반복
1, 5는 한번씩만 나옴
2, 3, 4는 2번씩 나옴
다친 손가락이 1, 5 -\> 다친 손가락 사용 횟수\*8 -\> 다친 손가락 직전에 셀 수 있는 만큼 추가 카운팅
다친 손가락이 2, 3, 4 -\> (다친 손가락 사용 횟수/2) \* 8 -\> 다친 손가락 직전에 셀 수 있는 만큼 추가 카운팅
결과 출력

C

```c
#include <stdio.h>

int main(){
    int hf; scanf("%d", &hf); // hf: Hurt Finger
    long long cnt; scanf("%lld", &cnt);
    int i = 0; long long res = 0;
    if (hf == 1 || hf == 5) {res += cnt * 8; cnt = 0;}
    else {res += 8 * (cnt / 2); cnt = (cnt % 2 == 0) ? 0 : 1;}

    while(1){
        int v = i % 8;
        if(v > 4) v = 8 - v;
        v++;
        if(v == hf) {
            if(cnt == 0) break; // 더 셀 수 없음
            else cnt--;
        }
        res++; i++;
    }
    printf("%lld", res);
    return 0;
}
```

Python

```python
hf = int(input())
cnt = int(input())
res = 0

if hf == 1 or hf == 5:
    res += cnt * 8
    cnt = 0
else:
    res += 8 * (cnt // 2)
    cnt = 0 if cnt % 2 == 0 else 1

i = 0
while True:
    v = i % 8
    if v > 4:
        v = 8 - v
    v += 1

    if v == hf:
        if cnt == 0:
            break
        else:
            cnt -= 1
    res += 1
    i += 1
    
print(res)
```
