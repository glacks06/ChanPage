[https://www.acmicpc.net/problem/1292](https://www.acmicpc.net/problem/1292)

반복문을 통해서 어떤 숫자 n을 n번 확인할 수 있도록 한다.
이 과정에서 입력 받은 값의 범위(자릿수)에 있다면 해당 값들을 결과(sum)에 더한다.

C

```c
#include <stdio.h>

int main() {
    int a, b; scanf("%d %d", &a, &b);
    int cnt = 0, sum = 0;
    for(int i = 1; i <= 45; i++){
        for(int j = 1; j <= i; j++){
            cnt++;
            if(cnt >= a && cnt <= b) sum += i;
    }}
    printf("%d", sum);
}
```

<br>

Python

```python
a, b = map(int, input().split())

sum = 0; cnt = 0
for i in range(1, 46):
    for j in range(i):
        cnt += 1
        if cnt >= a and cnt <= b: sum += i
print(sum)
```
