[https://www.acmicpc.net/problem/2490](https://www.acmicpc.net/problem/2490)

수(0혹은1) 4개를 입력받고, 값을 합산한 다음 결과가 0이면 D, 1이면 C, 2이면 B, 3이면 A, 4이면 E를 출력하여 문제를 해결하였다.

C

```c
#include <stdio.h>

int main() {
    int a, b, c, d, sum;
    for(int i = 0; i < 3; i++){
        scanf("%d %d %d %d", &a, &b, &c, &d);
        sum = a + b + c + d;
        
        switch(sum){
            case 0: 
                printf("D\n"); break;
            case 1:
                printf("C\n"); break;
            case 2:
                printf("B\n"); break;
            case 3:
                printf("A\n"); break;
            case 4:
                printf("E\n"); break;
        }
    }
}
```

Java

```java
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      for(int i = 0; i < 3; i++){
          int a = s.nextInt(), b = s.nextInt(), c = s.nextInt(), d = s.nextInt();
          int sum = a + b + c + d;
          switch(sum){
              case 0: System.out.println("D"); break;
              case 1: System.out.println("C"); break;
              case 2: System.out.println("B"); break;
              case 3: System.out.println("A"); break;
              case 4: System.out.println("E"); break;
            }
        }
    }
}
```
