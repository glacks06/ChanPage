[https://www.acmicpc.net/problem/2420](https://www.acmicpc.net/problem/2420)

주어진 두 값의 차를 구하기만 하면 되는 간단한 문제.

C

```c
#include <stdio.h>

int main() {
    long long n, m;
    scanf("%lld %lld", &n, &m);
    
    if(n - m >= 0) printf("%lld", n - m);
    else printf("%lld", (n - m) * -1);
}
```

<br>

Python

```python
n, m = input().split()
n = int(n)
m = int(m)

if(n - m >= 0):
    print(n - m)
else:
    print((n - m) * -1)
```

<br>

Java

```java
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      long n = s.nextLong(), m = s.nextLong();
      System.out.print((n - m < 0) ? -1*(n-m) : n-m);
    }
}
```
