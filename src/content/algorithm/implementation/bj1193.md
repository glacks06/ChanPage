분자 분모를 따로 보며 생각해야하는 문제

분자만 보면
1
1 2
3 2 1
1 2 3 4
5 4 3 2 1
...
줄바꿈 기준으로 맨 위부터 1층, 2층... 형태로 생각하면
n층은 1\~n까지의 수열이 있고
홀수층은 내림차순, 짝수층은 오름차순이다.

주어진X값에 -1, -2.. 를 하며 0이 되기 전까지 빼면서 몇층(v)인지 알아내고,
홀수층이면 v-(남은X + 1) 출력
짝수층이면 (남은 X) 출력

(분모는 분자의 계산에서 홀수층이면 짝수층 계산, 짝수층이면 홀수층 계산)

C

```c
#include <stdio.h>

int main() {
    int x; scanf("%d", &x);
    
    int v;
    for(int i = 1; ; i++){
        if(x - i <= 0){v = i; break;}
        x -= i;
    }
    if(v % 2 == 0) printf("%d/%d", x, v - x+1);
    else printf("%d/%d", v - x+1, x);
}
```

<br>

Python

```python
x = int(input())

i = 0; v = 0 # 층수
while 1:
    i += 1
    if x - i <= 0:
        v = i
        break
    x -= i

if v % 2 == 0: print('%d/%d' % (x, v - (x-1)))
else: print('%d/%d' % (v - (x-1), x))
```
