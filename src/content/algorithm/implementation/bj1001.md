[https://www.acmicpc.net/problem/1001](https://www.acmicpc.net/problem/1001)

주어진 입력값을 두 변수 a, b에 저장하고, 이 둘의 차를 출력하는 단순한 문제.

\---

C

```c
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", a-b);
}
```

<br>

Python

```python
a, b = input().split()
a = int(a)
b = int(b)
print("%d" % (a-b))
```

<br>

Java

```java
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      int a = s.nextInt(), b = s.nextInt();
      System.out.format("%d", a - b);
    }
}
```
