첫째 줄에 공백 0개와 별 N개, 둘째 줄에 공백 1개와 별 N-1개, 셋째 줄에 공백 2개와 별 N-2개... N번째 줄에 공백 N-1개와 별 1개를 출력하는 형식에서 (각 줄의 별의 개수) - 1 만큼을 추가로 출력하여 해결하였다.

C

```c
#include <stdio.h>

void prints(int v, int n) {
    for(int i = 0; i < v; i++) printf(" ");
    for(int i = 0; i < n + (n - 1); i++) printf("*");
    printf("\n");
}

int main() {
    int n; scanf("%d", &n);
    for(int i = n; i >= 1; i--) prints(n - i, i);
    return 0;
}
```

def prints(v, n):
    for _ in range(v): print(' ', end='')
    for _ in range(n + (n - 1)): print('*', end='')
    print()

n = int(input())
for i in range(n, 0, -1): prints(n - i, i)

Java

```java
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      int n = s.nextInt();
      
      for(int i = n; i > 0; i--){
          for(int j = n-i; j > 0; j--) System.out.print(" ");
          for(int j = i; j > 0; j--) System.out.print("*");
          for(int j = i-1; j > 0; j--) System.out.print("*");
          System.out.print("\n");
      }
    }
}
```
