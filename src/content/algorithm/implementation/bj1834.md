[https://www.acmicpc.net/problem/1834](https://www.acmicpc.net/problem/1834)

입력받은  n값으로 나누었을 때 나머지와 몫이 같은 모든 자연수를 구하고, 그 합을 출력하는 문제이다.

고등학교 수학을 하다보면 몫과 나머지를 수식으로써 표현하는 방법을 배운다.
(나누어지는 수) = (나누는 수)\*(몫) + (나머지)
이에 기반하여 수식을 생각해볼 수 있다.

만약 n이 10이라고 가정하면
어떤 수를 10으로 나누었을 때 나머지가 1이고, 몫도 1이 되려면 어떤 수는 11이 되어야 한다.
이는 10*1+1로 표현할 수 있다.
마찬가지로 22를 10으로 나누면 10*2+2로 표현할 수 있다.

결국 어떤 수를 n으로 나누었을 때 나머지와 몫이 v가 되려면 그 어떤 수는
n\*v+v 가 된다는 것을 알 수 있다.

따라서 v가 1\~n-1일때의 모든 n\*v+v를 더해준 후 그 결과를 출력함으로써 문제를 해결할 수 있다.

추가로,
n*v+v = v(n+1) 이면서 v가 1\~n-1일때를 모두 더해야 하므로
1*(n+1) + 2\*(n+1) +  2\*(n+1) +  ... (n-1)\*(n+1) = (((n-1)\*n) / 2) \* (n+1)
으로 정리할 수 있다.

C

```c
#include <stdio.h>

int main() {
    long long n; 
    scanf("%lld", &n);
    long long sum = (n + 1) * (n * (n - 1) / 2);
    printf("%lld", sum);
    return 0;
}
```
