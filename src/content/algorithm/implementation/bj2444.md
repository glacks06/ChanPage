별 찍기 5와 6을 결합한 형태. 별 찍기 5의 과정을 거치고, 별 찍기 6의 과정 중 가장 첫번째 줄을 출력하는 과정을 제외하여 출력하여 문제를 해결하였다.

C

```c
#include <stdio.h>

void prints(int v, int n) {
    for(int i = 0; i < v; i++) printf(" ");
    for(int i = 0; i < n + (n - 1); i++) printf("*");
    printf("\n");
}

int main() {
    int n; scanf("%d", &n);
    for(int i = 1; i <= n; i++) prints(n - i, i);
    for(int i = n - 1; i >= 1; i--) prints(n - i, i);
}
```

Java

```java
import java.util.Scanner;

public class Main {
    static void printStar(int space, int star){
        for(int i = 0; i < space; i++) System.out.print(" ");
        for(int i = 0; i < star; i++) System.out.print("*");
    }
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      int n = s.nextInt();
      
      for(int i = n; i > 0; i--){
          printStar(i-1, n-i+1); printStar(0, n-i);
          System.out.print("\n");
      }
      for(int i = 1; i < n; i++){
          printStar(i, n-i); printStar(0, n-1-i);
          System.out.print("\n");
      }
    }
}
```
