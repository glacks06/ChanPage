별 찍기 5와 6을 결합한 형태이다. 별 찍기 6의 과정을 먼저 거치고, 별 찍기 5의 과정을 거치면 된다. 이때 별 찍기 5의 과정에서 첫번째 과정은 제외해야 한다.

C

```c
#include <stdio.h>

void prints(int v, int n) {
    for(int i = 0; i < v; i++) printf(" ");
    for(int i = 0; i < n + (n - 1); i++) printf("*");
    printf("\n");
}

int main() {
    int n; scanf("%d", &n);
    for(int i = n; i >= 1; i--) prints(n - i, i);
    for(int i = 2; i <= n; i++) prints(n - i, i);
}
```

    for _ in range(v): print(' ', end='')
    for _ in range(n + (n - 1)): print('*', end='')
    print()

Java

```java
import java.util.Scanner;

public class Main {
    static void printStar(int space, int star){
        for(int i = 0; i < space; i++) System.out.print(" ");
        for(int i = 0; i < star; i++) System.out.print("*");
    }
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      int n = s.nextInt();
      
      for(int i = n; i > 0; i--){
          printStar(n-i, i); printStar(0, i-1);
          System.out.print("\n");
      }
      for(int i = n-1; i > 0; i--){
          printStar(i-1, n-i+1); printStar(0, n-i);
          System.out.print("\n");
      }
    }
}
```

function printStar(space, star){
    let res = ''
    for(let i = 0; i < space; i++) res += ' '
    for(let i = 0; i < star; i++) res += '*'
    return res
}
