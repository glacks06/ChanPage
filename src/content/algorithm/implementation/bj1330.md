[https://www.acmicpc.net/problem/1330](https://www.acmicpc.net/problem/1330)

주어진 입력값을 두 변수 a, b에 저장하고
a \> b 이면 '\>'를
a = b 이면 '=='를
a \< b 이면 '\<'를
출력하는 단순한 문제.

C

```c
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    
    if(a > b) printf(">");
    else if(a < b) printf("<");
    else printf("==");
}
```

<br>

Python

```python
a, b = input().split()
a = int(a)
b = int(b)
if a > b:
    print(">")
elif a < b:
    print("<")
else:
    print("==")
```

<br>

Java

```java
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      int a = s.nextInt(), b = s.nextInt();
      if(a > b) System.out.print(">");
      else if(a < b) System.out.print("<");
      else System.out.print("==");
    }
}
```
