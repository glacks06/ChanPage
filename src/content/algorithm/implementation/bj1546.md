주어진 점수 중 최댓값을 찾으면서 모든 점수의 합(sum)을 구하고 (반복문)

{(모든 점수의 합) / (최댓값)} / (과목 수) 을 출력

C

```c
#include <stdio.h>

int main() {
    int n; scanf("%d", &n);
    int tempN = n, sum = 0, mx = 0;
    while(n--){
        int v; scanf("%d", &v);
        sum += v;
        if(v > mx) mx = v;
    }
    printf("%.2f", ((sum*100) / (float)mx) / tempN);
}
```

<br>

Python

```python
from sys import stdin

n = int(input())

sum = 0; mx = 0
score = map(int, stdin.readline().split())
for i in score:
    sum += i
    if i > mx: mx = i
print(((sum*100) / mx) / n)
```
