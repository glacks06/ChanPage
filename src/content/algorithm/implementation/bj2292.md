[https://www.acmicpc.net/problem/2292](https://www.acmicpc.net/problem/2292)

주어진 값(N)이 있는 방이 1번 방으로부터 몇번 떨어져 있는지 확인하는 문제이다. 

이 '몇번'은 '몇껍질'만큼 떨어져 있느냐로 생각해볼 수 있다. 
2\~7은 1껍질만큼 떨어져 있고, 8\~19는 2껍질만큼 떨어져있는 방식이다. 
방의 개수는 한 껍질단위마다 방의 개수가 6개씩 증가한다. 

따라서 N이 1보다 작아질때까지 N에 6, 12, 18, 24 .... 만큼씩 빼준 후 뺀 횟수를 출력하여 문제를 해결하였다.

C

```c
#include <stdio.h>

int main() {
    int n; scanf("%d", &n);
    int i = 0, cnt = 1; 
    while(n > 1){
        i += 6;
        n -= i;
        cnt++;
    }
    printf("%d", cnt);
}
```

<br>

Python

```python
n = int(input())

i = 0; cnt = 1
while n > 1:
    i += 6
    n -= i
    cnt += 1
print(cnt)
```
