[https://www.acmicpc.net/problem/2475](https://www.acmicpc.net/problem/2475)

각 수를 입력받고, 그 수의 제곱을 합산(sum)한 뒤 10으로 나눈 나머지를 출력하여 문제를 해결하였다.

C

```c
#include <stdio.h>

int main() {
    int sum = 0, num;
    for(int i = 0; i < 5; i++) {
        scanf("%d", &num);
        sum += num * num;
    }
    sum %= 10;
    printf("%d", sum);
}
```
