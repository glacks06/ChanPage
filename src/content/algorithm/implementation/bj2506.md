[https://www.acmicpc.net/problem/2506](https://www.acmicpc.net/problem/2506)

n을 입력받고 반복문을 통해 n번 입력받으며 연산을 하였다. 입력값이 1이면 점수 변수(코드상에서의 v. 처음에 0으로 시작)에 +1을 하고, 최종 결과(score)에 점수를 저장하여 문제를 해결하였다.

C

```c
#include <stdio.h>

int main() {
    int n; scanf("%d", &n);
    int score = 0, v = 0, a;
    while(n--){
        scanf("%d", &a);
        v = (a == 1) ? ++v : 0;
        score += v;
    }
    printf("%d", score);
}
```

Python

```python
input()
a = map(int, input().split())
score = 0; v = 0

for i in a:
    v = 0 if i == 0 else v + 1
    score += v
print(score)
```

Java

```java
import java.util.Scanner;

public class Main{
    public static void main(String args[]) {
      Scanner s = new Scanner(System.in);
      int n = s.nextInt(), score = 0, plus = 0;
      for(int i = 0; i < n; i++){
          int v = s.nextInt();
          if(v == 1) score += ++plus;
          else plus = 0;
      }
      System.out.print(score);
    }
}
```
