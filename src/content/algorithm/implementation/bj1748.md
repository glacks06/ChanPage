[https://www.acmicpc.net/problem/1748](https://www.acmicpc.net/problem/1748)

숫자 하나하나 자릿수를 계산하는 것은 시간이 걸리므로
주어진 값 N의 자릿수 이전 자릿수의 숫자들을 미리 연산한다.(반복문)
(일의자리 숫자들은 9개, 십의자리 숫자들은 90개 ....)
그 다음 남은 숫자의 개수와 자릿수를 곱하여서 추가로 더한 후
출력한다.

C

```c
#include <stdio.h>

int main(){
    int n; scanf("%d", &n);
    int i = 1, j = 1;
    long long res = 0; // 결과값이 커질 수 있으므로 long long 사용
    
    // N보다 작은 자릿수들 (9, 90, 900...) 먼저 계산
    while(n >= i * 10){
        res += (9LL * i) * j;
        i *= 10;
        j++;
    }
    
    // 남은 자릿수 계산
    res += (long long)(n - (i - 1)) * j;
    printf("%lld", res);
    
    return 0;
}
```

<br>

Python

```python
n = int(input())
i = 1 # 1, 10, 100...
j = 1 # 자릿수 (1, 2, 3...)
res = 0

# N보다 작은 자릿수의 숫자들 개수 미리 합산
while n >= i * 10:
    res += (9 * i) * j
    i *= 10
    j += 1

# 마지막 남은 자릿수만큼의 숫자들 합산
res += (n - (i - 1)) * j
print(res)
```

계산 방식이 자릿수가 늘어날 때마다 기하급수적으로 늘어나는 숫자의 개수($9 \times 10^{n-1}$)를 활용하는 효율적인 방식이네요\!
